# MikiJapan Verify

หน้า Verify แบบ frontend-only สำหรับแสดงโลโก้ร้านและข้อความรอตรวจสอบข้อมูล

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS ผ่าน `@tailwindcss/vite`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

โปรเจกต์นี้ไม่มีการเชื่อมต่อ backend, API หรือ LIFF
